## Tecnologías usadas

- HTML
- CSS
- JavaScript vanilla con fetch()

## Cómo correr el proyecto

1. Clonar el repositorio:
git clone https://github.com/Samuuu132/f1-frontend.git
2. Abrir el archivo `index.html` en el navegador
   o usar Live Server en el editor de código

## Funcionalidades

- Ver todos los pilotos en tarjetas
- Agregar un piloto nuevo con foto desde el dispositivo
- Editar un piloto existente
- Eliminar un piloto

## Challenges implementados

- ✅ Subida de imágenes desde el dispositivo (Base64)
- ✅ Validación de tamaño máximo de 1MB en imágenes
- ✅ Diseño temático de Fórmula 1

## Reflexión

Desarrollé el cliente utilizando únicamente HTML, CSS y JavaScript vanilla con
fetch(), lo que me permitió comprender en profundidad cómo funciona la
comunicación entre un cliente y una API REST sin depender de frameworks.
El mayor desafío fue manejar la conversión de imágenes a Base64 directamente
desde el navegador. Utilizaría este enfoque nuevamente para proyectos donde
se requiera simplicidad y control total sobre el código.
